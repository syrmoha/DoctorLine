## 📝 عن المشروع (About the Project)
**"دكتور لاين"** هي منصة إلكترونية وواجهة تعريفية لعيادة طب أسنان. تهدف المنصة إلى تسهيل الوصول لخدمات العيادة، استعراض خبرات الأطباء، قراءة آراء المرضى، وإمكانية التواصل وحجز المواعيد بكل سهولة عبر واجهة مستخدم (UI) جذابة وسريعة التصفح. 

## ✨ الميزات والتفاصيل (Features)
- 🎨 **تصميم عصري واحترافي:** ألوان متناسقة، تركيز على راحة العين (UI/UX) باستخدام تموجات حديثة وانتقالات ناعمة.
- 📱 **متجاوب بالكامل (Fully Responsive):** يدعم جميع أحجام الشاشات (الهواتف المحمولة، الأجهزة اللوحية، وشاشات سطح المكتب).
- 🌐 **ثنائي اللغة (Bilingual):** يوفر المشروع نسخة عربية بدعم كامل للـ RTL، ونسخة إنجليزية بدعم للـ LTR.
- ⚡ **تفاعل ديناميكي:** 
  - قائمة تنقل جانبية مخصصة للموبايل (Mobile Menu) مع استبدال تفاعلي لأيقونة القائمة.
  - قسم تفاعلي للأسئلة الشائعة (FAQ) بنظام (Accordion).
  - انتقال سلس وهادئ بين أقسام الصفحة (Smooth Scrolling).
- 📍 **خريطة تفاعلية:** تضمين خريطة جوجل (Google Maps) لموقع العيادة لتسهيل وصول المرضى.
- 🚀 **أداء عالي:** مبني باستخدام لغات الويب الأساسية بدون أي مكتبات ثقيلة ليقدم سرعة تحميل فائقة.

## 🛠️ التقنيات المستخدمة (Technologies Used)
تم بناء المشروع بشكل نقي (Vanilla) للحصول على أفضل أداء وأسرع وقت تحميل للموقع:
- **[HTML5]**: الهيكلة الأساسية للصفحات.
- **[CSS3]**: تنسيقات الموقع بالكامل (Flexbox, Grid, CSS Variables, Media Queries & Animations).
- **[Vanilla JavaScript]**: لإضافة التفاعلات والديناميكية للموقع (القوائم، الأسئلة الشائعة).
- **[FontAwesome 6.4.0]**: لأيقونات الموقع.
- **[Google Fonts]**: إضافة خطوط عصرية (Cairo للنسخة العربية، وتشكيلة متناسقة للنسخة الإنجليزية).

## 🚀 كيفية التشغيل (Setup & Installation)
بما أن المشروع عبارة عن صفحات ويب أساسية (Static Pages)، فهو لا يحتاج إلى أي أدوات معقدة أو بيئات تشغيل:
1. قم بتحميل الملفات كـ `.ZIP` أو استنسخ المستودع:
   ```bash
   git clone https://github.com/YourUsername/DoctorLine-main.git
   ```
2. قم بفك الضغط والدخول إلى مجلد المشروع.
3. قم بفتح ملف `index.html` لعرض النسخة العربية أو `en.html` في متصفح الويب الخاص بك.
